Descripción

En el meetup realizamos dos actividades.

La primera actividad consistió en aprender como crear un Addon de ember-cli desde cero hasta su publicación en el repositorio NPM.

Para la segunda actividad creamos un Pull Request para el proyecto open source ember-cli-page-object con el cuál aprendimos a crear generadores (blueprints) de ember-cli.

Ambas actividades fueron presentadas por Santiago.

Recursos

  • Notas de la actividad ¿Cómo crear un addon para ember-cli? por Santiago
  • Notas de la actividad Agregar generadores a ember-cli-page-object por Santiago
  • Código del addon que se construyó
  • Link al Pull request que se creó
