Actividades

Deploys en Ember

Marcelo Dominguez y Juan Azambuja van a contar sobre deployment con Ember CLI y cómo notificar a los clientes de que una nueva versión está disponible.

Linters, code conventions and continuous integration

Julio Barrios y Luis Ferreira van a presentar herramientas para mantener el código limpio, validando que cumpla con estándares de codificación y tenga sus dependencias al día. Luego vimos una configuración de continuous integration para integrar todas estas buenas prácticas.
